# Changelog

## 0.1.0 (2026-03-15)

- Initial release
- Primitive type guards (isString, isNumber, isBoolean, etc.)
- Structural type guards (isArray, isObject, isMap, isSet, etc.)
- Value guards (isNonEmpty, isPositive, isInteger, etc.)
- Composable guards (isOneOf, isArrayOf, isObjectWith, isTupleOf)
- Assert function for throwing on guard failure
